Principais formatações

Negrito / destaques

Nesta frase temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase temos um termo em destaque usando a tag <strong> (semântica).

Itálico / destaques

Nesta frase temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase temos um termo em ênfase usando a tag <em> (semântica).

Texto marcado

Podemos criar um texto marcado usando a tag <mark>

E no outro parágrafo temos outro texto marcado no final

Texto grande e pequeno

Estamos criando um texto grande e um pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag <del> (semântica).

Texto inserido

Podemos marcar um texto como inserido para da uma ênfase, para indicar que ele foi adicionado depois, com a tag <ins> (semântica).

Existe também o sublinhado com a tag <u> (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3, usando a tag <sup> (semântica).

Texto subscrito

Para inserir coisas do tipo H2O, usando a tag <sub> (semântica).
